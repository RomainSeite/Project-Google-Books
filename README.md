* **Data Set** [`DatasetGoogleBookAPI.json`](https://github.com/RomainSeite/Project-Google-Books/blob/main/DatasetGoogleBookAPI.json) : Data Set for the collection. There are 11 iterations : for every request, up to 6 correct scenarios, and 5 incorrect ones (3 error 400, error 404 then error 503). Note that `error 429: too many requests` absolutely works and limits the testability of the API :D
